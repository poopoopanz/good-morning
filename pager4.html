Loops

Loops allow us to repeat actions multiple times. Here are examples of the most common types of loops:

For Loop

# For loop to iterate through a list
numbers = [1, 2, 3, 4, 5]
for number in numbers:
    print(number)

While Loop

# While loop that runs as long as a condition is true
count = 0
while count < 5:
    print(count)
    count += 1
