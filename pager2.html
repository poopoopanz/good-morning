Python Basics

Let's start with the very basics of Python. Here are some simple examples:

Variables

# This is how we define variables in Python
name = "John"
age = 25
print(name, age)

Data Types

# Python has various data types:
# String, Integer, Float, List, Dictionary, etc.
my_string = "Hello, Python!"
my_number = 10
my_float = 20.5

Basic Input/Output

# Taking input from the user
name = input("What's your name? ")
print(f"Hello, {name}!")
